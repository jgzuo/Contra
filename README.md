# 🎮 魂斗罗 - Contra

一个经典的横向卷轴射击游戏，采用纯 HTML5 Canvas 实现，带来怀旧的街机游戏体验。

## ✨ 游戏特性

### 核心玩法
- 🔫 **经典射击** - 横向卷轴射击游戏机制
- 👾 **多样敌人** - 面对不同类型的敌人挑战
- 💥 **武器系统** - 多种武器选择和升级
- 🎯 **精准操作** - 流畅的角色控制和射击

### 视觉设计
- 🌌 **霓虹风格** - 赛博朋克风格的视觉效果
- ✨ **动态背景** - 滚动的背景和光效
- 🎨 **流畅动画** - 精美的游戏动画
- 💫 **粒子特效** - 爆炸和射击特效

## 🎮 操作方式

| 按键 | 功能 |
|------|------|
| ← → 或 A D | 左右移动 |
| ↑ 或 W | 跳跃 |
| Z 或 J | 射击 |
| R | 重新开始 |

## 🚀 快速开始

### 在线游玩
访问 GitHub Pages: **https://jgzuo.github.io/Contra/**

### 本地运行
1. 克隆仓库：
   ```bash
   git clone https://github.com/jgzuo/Contra.git
   cd Contra
   ```

2. 直接在浏览器中打开 `index.html`
3. 或使用本地服务器：
   ```bash
   # 使用 Python
   python -m http.server 8000

   # 访问
   http://localhost:8000
   ```
